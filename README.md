# Laravel Blade Sugar

Some useful sugar for your Laravel Blade templates to keep 'em DRY and clean.

## Usage

In your ```config/app.php```, add the service provider:

```php
'providers' => [

    BC\Laravel\LaravelBladeSugar\ServiceProvider::class,

],
```
